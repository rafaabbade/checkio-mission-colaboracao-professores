Dada uma lista de pares de colaboração entre professores, crie um dicionário onde cada chave é o nome de um professor e o valor correspondente é uma lista de professores com quem ele colabora. Cada par na lista representa uma colaboração entre dois professores. O objetivo é mapear essas relações de forma clara e organizada.

Entrada Uma lista de pares (tuplas) representando relações de colaboração entre professores.

Saída: Um dicionário onde cada chave é o nome de um professor e cada valor é uma lista de outros professores com quem ele colabora.
